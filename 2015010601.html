创建拥有phonegap库的iOS项目

第一步：下载phonegap

下载phonegap是再简单不过的事，控制台的一句话就全部搞定 sudo npm install -g phonegap

第二步：创建一个phonegap的项目

进入一个你想创建的目录，输入以下命令： phonegap create forgithub cd forgithu ionic build ios 这样三句分别创建了phonegap项目，然后进入项目目录，再基于创建的phonegap项目创建了一个可以在iOS平台上运行的helloword项目。创建 进入iOS打开helloWord项目，直接run有下图。
